Initializing a git repository

If you want to add a file to stage a commit, you need to initialize a repository in that exisiting project directory.

To do this, you need to go to project's directory and enter the following command:
git status

We are already in project folder. Go ahead and hit the command.

Adding files

There is a file in the project directory. Check the status again and you will know what I mean.
git status

After checking status, you can see there is an untracked file named cat.txt. To start tracking it, we need to add it to the staging area. You are expected to do this by following command
git add cat.txt
