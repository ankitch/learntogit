Creating & switching branch

When you have new features to add to program but do not want to break your main program, git gives you a feature called branching. Branch is a separate copy of main program which your commits to it does not affect the master branch (your main program).

To create a new branch, enter the following command:
git branch insect

To switch to the branch, enter the following command:
git checkout insect
