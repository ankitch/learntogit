Pusing & pulling to and from remote repository

Now that you hae successfully added a remote repository, it is time to push changes to the master branch of remote host so that everyone connected to remote host can see and pull changes

To do this, enter the following command:
git push -u origin master

Likewise, if others have pushed changes to the remote host, you need to pull them to your local repository so you can start working on it. To pull, enter the following command:
git pull

We are already in project folder. Go ahead and hit the command.
